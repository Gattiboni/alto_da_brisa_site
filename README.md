## O Projeto

O **Alto da Brisa** é um projeto de vida comunitário localizado em **Sapucaí-Mirim, MG** (região de Gonçalves, Serra da Mantiqueira), sobre a Gleba 7 do Sítio São Miguel — 20.000 m² de área privativa, com presença de nascente própria e APP.

O empreendimento reúne três famílias que dividem não apenas o terreno, mas uma visão de como se quer viver: com natureza, autonomia, tecnologia e intenção.
